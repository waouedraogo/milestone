Course Description

Mobile Web Development

Define the basic terms and explain the concepts related to Mobile Application Development. Examine the Eclipse & Xcode SDKs along with HTML, CSS and JavaScript programming language. Configure the Adobe Dreamweaver and Eclipse SDK for Windows Operating System. Design and develop basic mobile web & hybrid native applications that conform to current standards.

Apply basic design principles when designing mobile applications. Demonstrate how to incorporate text and database elements into a mobile application. Create and optimize multimedia elements to be used for mobile applications. Create animated elements, physical interaction, and relationships in mobile application objects. Demonstrate how to deploy Android and iOS mobile applications using Phone Gap. Explain the distribution and marketing process related to mobile application marketplaces.
